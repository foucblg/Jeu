Inclusif, le jeu

2. Analyse de l'inclusivite de votre service numérique

Le but de cette étape est de vous permettre d’analyser l’inclusivité globale actuelle de votre service numérique.

Pour cela, vous allez répondre à une dizaine de questions portant sur différents domaines d’application de l’inclusivité numérique.
À la fin de cette étape, vous aurez une vision d’ensemble sur l’inclusivité de votre service.

A savoir : un outil d’évaluation détaillée est disponible dans la ressourcerie du jeu, pour évaluer plus finement votre service après l’application des bonnes pratiques
